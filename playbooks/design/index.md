# Design Playbooks

Use these playbooks when UI, brand, product feel, or presentation quality
matters.

## Read Order

1. Start at root `playbooks/index.md`.
2. Use this index to pick the smallest design playbook.
3. Load only the selected playbook and references it names.

## Playbooks

- `taste-calibration.md`: Establish the design read, aesthetic direction,
  design-system choice, and anti-default guardrails before UI work.
- `ui-taste-review.md`: Review UI for taste, clarity, states, accessibility,
  motion, content quality, and common AI-design defaults.
