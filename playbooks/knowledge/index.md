# Knowledge Playbooks

Use these playbooks to save, restore, and maintain reusable project knowledge.

## Read Order

1. Start at root `playbooks/index.md`.
2. Use this index to pick the smallest knowledge playbook.
3. Load only the selected playbook and references it names.

## Playbooks

- `continue-work-discovery.md`: Delegate resume discovery before continuing
  ticket, product, milestone, or trace work.
- `context-restore.md`: Restore saved work context before continuing.
- `context-save.md`: Save current work context for handoff or resume.
- `learn.md`: Capture reusable lessons without bloating root instructions.
